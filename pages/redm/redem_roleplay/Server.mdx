# Server

 `getPlayerFromId` or `playerModule`
```lua
local user = exports.redem_roleplay:getPlayerFromId(source)
local user = exports.redem_roleplay:playerModule(source)
```

## Function
```lua
user.setFirstname(name)
user.setLastname(name)
user.setJob(job)
user.setJobgrade(grade)
user.setOrigin(origin)
user.kick(reason)
```
